# Тестовые запросы для консольного приложения
Для примера тут расписано несколько вариантов и их результаты, чтобы было проще понять как состовлять запросы.

## Вариант 1:
**Параметры:**
- Район: **Moscow**
- Период: **2024-10-20 00:00:00** - **2024-10-25 23:59:59**
- Мин число заказов: **2**

**Результат:**
1. Номер заказа: 1, Вес: 7.2, Район: Moscow, Время доставки: 2024-10-20 15:30:00
2. Номер заказа: 5, Вес: 6.7, Район: Moscow, Время доставки: 2024-10-24 09:20:00
3. Номер заказа: 33, Вес: 3.6, Район: Moscow, Время доставки: 2024-10-24 09:50:00

---
## Пример 2:
**Параметы:**
- Район: **Vladivostok**
- Период: **2024-10-20 00:00:00** - **2024-10-26 23:59:59**
- Мин число заказов: **2**

**Результат:**
Должен быть пуст, в указанном диапазоне только один заказ.

---
## Пример 3:
**Параметры:**
- Район: **Kazan**
- Период: **2024-10-22 00:00:00** - **2024-10-28 23:59:59**
- Мин число заказов: **3**

**Результат:**

3. Номер заказа: 40, Вес: 9.7, Район: Kazan, Время доставки: 2024-10-22 12:50:00

## Пример 4:
**Параметры:**
