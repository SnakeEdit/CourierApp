# Тестовые запросы для консольного приложения
Для примера тут расписано несколько вариантов и их результаты, чтобы было проще понять как состовлять запросы.

## Вариант 1:
**Параметры:**
- Район: **Moscow**
- Время первой доставки **2024-10-24 09:00:00**

**Результат:**
- 5,6.7,Moscow,2024-10-24 09:20:00

---
## Пример 2:
**Параметы:**
- Район: **Kazan**
- Время первой доставки: **2024-10-22 12:00:00**

**Результат:**
Должен быть пуст, в диапазоне 30 минут, с 12:00 до 12:30, нет заказов.

---
## Пример 3:
**Параметры:**
- Район: **Sochi**
- Время первой доставки: **2024-10-28 19:00:00**

**Результат:**
Должен быть пуст, в диапазоне 30 минут, с 19:00 до 19:30, нет заказов.
Заказ - 28,7.4,Sochi,2024-10-28 19:55:00 не попадает в интервал.

---

## Пример 4:
**Параметры:**
- Район: **Tula**
- Время первой доставки: **2024-10-26 08:30:00**

**Результат:**
